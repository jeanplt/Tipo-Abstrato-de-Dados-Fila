# Tipo-Abstrato-de-Dados-Fila
Este repositório contém um programa modular em C que implementa e testa uma TAD Fila, utilizando a política FIFO (First In First Out) com uma lista ligada com nodo cabeça.

O programa modular que cria e testa a TAD Fila é composto pelos seguintes arquivos:

    fila.h: protótipos das funções, que criam, modificam e destroem a fila. Este arquivo foi desenvolvido pelos professores do departamento de informática da Universidade Federal do Paraná, sendo eles Andre Ricardo Abed Gregio, Marcos Alexandre Castilho e Luis Carlos Erpen de Bona.
    fila.c: implementação de todas as funções existentes em fila.h, feito por Jean Pablo Lopes Teixeira.
    testa_filha.c: desenvolvido por Marcos Alexandre Castilho, testa as implementações de fila.c verificando se a fila é criada corretamente, se os dados são obtidos sem problemas de ponteiros, se a fila manipula corretamente seus dados no topo (nodo cabeça) e se a fila entende que está vazia ou não.

Por fim, este programa tem o propósito apenas de testar a criação da fila, mas pode ser adaptado para utilizar a fila em outros contextos que envolvam utilização de lista ligada com nodo cabeça.
